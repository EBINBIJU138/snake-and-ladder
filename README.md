# 🐍 Snake and Ladder Game

A fun and colorful **Snake and Ladder game** made for kids and casual players. Roll the dice, climb ladders, avoid snakes, and race to reach square 100!

## 🎮 Game Features

- Classic Snake and Ladder gameplay
- Colorful game board
- Dice-based movement
- Snakes send players down
- Ladders help players climb up
- Simple and beginner-friendly

## 🖼️ Game Preview

![Snake and Ladder Game](./background.png)

## 🚀 How to Play

1. Start from square 1.
2. Roll the dice.
3. Move forward based on the dice number.
4. If you land on a ladder, climb up.
5. If you land on a snake, slide down.
6. First player to reach 100 wins.
