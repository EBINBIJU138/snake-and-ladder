# 🐍 Snake and Ladder

A classic **Snake and Ladder** game developed using **Unity** and **C#**, featuring custom-designed game assets created with **Adobe Photoshop** and **Blender**. Players roll the dice, climb ladders, avoid snakes, and race to reach the final tile before their opponents.

---

## 🎮 Features

* Classic Snake and Ladder gameplay
* Smooth dice rolling mechanics
* Interactive player movement
* Snakes and ladders system
* Custom-designed game board
* Optimized UI and game assets
* Easy-to-play controls
* Scalable project architecture

---

## 🛠️ Technologies Used

### Unity

Used as the primary game engine for developing gameplay, scene management, animations, and user interactions.

### C#

Used for implementing game logic, player movement, dice mechanics, turn management, and win conditions.

### Adobe Photoshop

Used for creating:

* UI elements
* Board textures
* Icons
* 2D game assets
* Promotional graphics

### Blender

Used for:

* 3D modeling
* Asset creation
* Environment props
* Game-ready model optimization

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/snake-and-ladder.git
```

2. Open the project in Unity Hub.
3. Load the project using the appropriate Unity version.
4. Open the Main Scene.
5. Press **Play** to start the game.

---

## 🎲 How to Play

1. Start from tile **1**.
2. Roll the dice.
3. Move forward according to the dice value.
4. Climb ladders to advance faster.
5. Avoid snakes that send you backward.
6. Reach tile **100** to win the game.
